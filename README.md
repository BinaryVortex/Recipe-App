# Recipe App

A simple, responsive Recipe App built with HTML, CSS and JavaScript.

![Recipe App Screenshot](./Screenshot%202024-04-29%20133502.png)

## Overview

This project is a lightweight front-end application that demonstrates a clean and user-friendly interface for browsing recipes. It is intended as a small portfolio project or learning exercise for working with vanilla JavaScript and modern CSS techniques.

## Features

- Clean, responsive UI built with HTML & CSS
- Interactive behavior implemented with plain JavaScript
- Easy to run locally — no build step or dependencies required

## Demo / Usage

To run the app locally:

1. Clone the repository:

   git clone https://github.com/BinaryVortex/Recipe-App.git

2. Open the project folder and open `index.html` in your browser. For a better development experience you can use a static file server such as VS Code Live Server or Python's built-in server:

   - Using Python 3:
     ```
     cd Recipe-App
     python -m http.server 8000
     ```
     Then open http://localhost:8000 in your browser.

## Project Structure

- `index.html` - The main HTML page
- `css/` - Stylesheets
- `js/` - JavaScript files
- `assets/` - Images and other static assets (including the screenshot)
